Exercices de la semaine

Exercice simple

Vous trouverez à l'url https://exercices-web.herokuapp.com/exos/long/double/4 un web service qui double la valeur du nombre qu'on lui passe.

Tu dois démontrer un appel avec succès à ce web service dans une application Android depuis l'interface graphique.

Par exemple, une activité avec un EditText pour le nombre et un bouton pour déclencher l'appel. Au retour de la valeur (doublée), celle-ci est affichée dans un Toast.

Exercice Objet complexe

Créez un projet appelé Complexe. Le projet doit partir une activité qui doit aller récupérer des données ici (remplacez whippsie par votre prénom dans l'URL). Les champs récupérés doivent être affichés dans l'activité principale.

Exercice Listes

Créez un projet appelé Listes. Le projet doit partir une activité coupée en 2 dans le sense de la hauteur. Chaque moitié est occupée par un recyclerView qui sera peuplée par des appels suivants:

  • https://exercices-web.herokuapp.com/exos/long/list https://exercices-web.herokuapp.com/exos/long/list renvoie une liste de Long, chaque élément dans le recyclerView est un simple champ text où vous affichez la valeur du nombre
  • https://exercices-web.herokuapp.com/exos/truc/list renvoie une liste d'objets complexes. Chaque item du recyclerView doit afficher les 2 champs simple ainsi que la taille de la liste

Exercice Github API

Créez un projet appelé Repos dans lequel vous créerez un service simple en GET sur l'URL https://api.github.com/users/jorisdeguet/repos
